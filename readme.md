## Get started

```bash
git clone https://github.com/danielsteman/.dotfiles.git ~/.dotfiles
sh ~/.dotfiles/unix/install.sh ; sh ~/.dotfiles/unix/create-terminal-setup.sh


source ~/.zshrc
```
